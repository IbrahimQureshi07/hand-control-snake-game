# 🎮 Hand-Controlled Snake Game

## Quick Start (5 Minutes)

### 1. Install Libraries
Terminal/Command Prompt kholo aur yeh command run karo:
```bash
pip install -r requirements.txt
```

### 2. Run Game
```bash
python snake_game_tkinter.py
```

### 3. Hand Control Mode On Karo
- `snake_game_tkinter.py` file kholo
- Line 11 par jao
- `CONTROL_MODE = 'keyboard'` ko `CONTROL_MODE = 'hand'` kar do
- Save karo aur phir se run karo

## 🎯 How to Play

### Keyboard Mode:
- Arrow keys se snake move karo

### Hand Mode:
- Camera ke saamne khade ho jao
- Index finger ko move karo:
  - **Up** = Snake up jayega
  - **Down** = Snake down jayega
  - **Left** = Snake left jayega
  - **Right** = Snake right jayega

## 📁 Files

- `snake_game_tkinter.py` - Main game file
- `requirements.txt` - Libraries list
- `SETUP_GUIDE.md` - Detailed setup instructions
