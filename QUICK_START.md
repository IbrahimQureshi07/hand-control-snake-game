# 🎮 Quick Start Guide - Snake Game

## ✅ Good News! Game Ready Hai - Koi Installation Nahi Chahiye!

Main ne **2 versions** banaye hain:

### Version 1: tkinter Version (✅ RECOMMENDED - No Installation!)
**File:** `snake_game_tkinter.py`

**Kyun use karein?**
- ✅ Python ke saath built-in hai (kuch install nahi karna)
- ✅ Direct run hoga
- ✅ Same features (arrow keys, food, score, etc.)

**Run karne ke liye:**
```bash
python snake_game_tkinter.py
```

---

### Version 2: pygame Version (Agar pygame install ho jaye)
**File:** `snake_game.py`

**Kyun use karein?**
- Better graphics
- Smoother animation
- Professional look

**Run karne ke liye:**
```bash
pip install pygame  # Pehle install karo
python snake_game.py
```

---

## 🚀 Abhi Ke Liye - tkinter Version Use Karo!

### Step 1: Terminal Kholo
- `Win + R` → `cmd` → Enter

### Step 2: Folder Mein Jao
```bash
cd "C:\Users\User\Desktop\hand snake game task"
```

### Step 3: Game Run Karo
```bash
python snake_game_tkinter.py
```

**Bas! Game start ho jayega!** 🎉

---

## 🎮 Game Features (Dono Versions Mein Same)

✅ **Snake Movement**: Arrow keys se snake move hota hai  
✅ **Food (Red Balls)**: Random jagah par red food appear hota hai  
✅ **Eating**: Snake food ko khaata hai  
✅ **Growth**: Food khane ke baad snake lamba hota hai  
✅ **New Food**: Har baar food khaane ke baad naya food alag jagah par aata hai  
✅ **Score**: Har food par 10 points milte hain  
✅ **Game Over**: Snake apne aap se takraaye to game over  
✅ **Restart**: 'R' key press karke restart kar sakte ho  

---

## 🎯 Controls

- **↑ (Up Arrow)**: Snake up jayega
- **↓ (Down Arrow)**: Snake down jayega
- **← (Left Arrow)**: Snake left jayega
- **→ (Right Arrow)**: Snake right jayega
- **R Key**: Game over ke baad restart karne ke liye
- **Close Window**: Game band karne ke liye

---

## 🔧 Control Mode Kaise Change Karein?

### tkinter Version: `snake_game_tkinter.py`
**Line 8:**
```python
CONTROL_MODE = 'keyboard'  # YAHAN CHANGE KARO
```

### pygame Version: `snake_game.py`
**Line 8:**
```python
CONTROL_MODE = 'keyboard'  # YAHAN CHANGE KARO
```

**Options:**
- `'keyboard'` = Arrow keys (abhi ke liye)
- `'hand'` = Hand gestures (baad mein camera ke saath)
